Shadow Fleet Tracker — Maritime Intelligence Platform

Predictive OSINT Platform: Mapping the Hidden Russian ‘Shadow Fleet’

Year: 2025

Technologies: Python, Flask/FastAPI, GeoPandas, Leaflet.js, JavaScript, WebSockets, AISStream.io, TeleGeography GeoJSON

Overview

The Shadow Fleet Tracker is a real-time maritime intelligence platform that detects and visualises suspicious vessel activity worldwide using AIS (Automatic Identification System) metadata. It identifies vessels operating under flags of convenience or loitering near critical undersea infrastructure such as submarine cables, contributing to open-source maritime situational awareness.

Key Features

  • Real-time AIS data ingestion and anomaly detection for covert activity patterns.
  • Geofencing alerts for proximity to critical maritime zones and infrastructure.
  • Interactive Leaflet.js dashboard with live vessel tracking and classification filters.
  • Integration-ready architecture for satellite and sanctions datasets to extend OSINT coverage.

Technical Details

The backend is built with Python using Flask/FastAPI and GeoPandas for spatial analysis. Real-time AIS streams are handled via WebSockets and visualised through Leaflet.js. Data layers incorporate TeleGeography’s undersea cable GeoJSON datasets and public vessel registries. Its modular structure supports future AI-based anomaly detection and satellite imagery fusion.

Collaboration & Methodology

This project was coded collaboratively. I was responsible for the system architecture, backend design, and structural front-end, while Bradley Ashton developed advanced UI and data visualisation features. We worked via Discord and Slack, using a tight SDLC with a loose iterative methodology to allow rapid testing and refinement during development.

Rationale & Purpose

This platform explores how open-source maritime metadata can be used to identify covert Russian “shadow fleet” activity and monitor potential threats to UK undersea infrastructure. It highlights the role of data-driven intelligence and machine learning-enhanced OSINT in supporting maritime resilience, national security, and situational awareness.
