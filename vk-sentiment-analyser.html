VKontakte Sentiment Analyser

Live VKontakte Media Monitoring Using NLP and Sentiment Analysis of Russian News Agency Posts

Year: 2025

Technologies: Python, Flask, Hugging Face Transformers, VK API, JavaScript, Bootstrap 5

Overview

The VKontakte Sentiment Analyser builds on the Russian Sentiment Analysis Bot by applying natural language processing to live Russian social media posts. It connects to the VKontakte API to gather recent headlines from Russian news agencies and analyses sentiment using the RuBERT model from Hugging Face. The system provides real-time insights into how narratives surrounding the Ukraine conflict are being presented inside Russia.

Key Features

  • Connects directly to the VKontakte API to fetch live news posts from verified Russian media outlets.
  • Uses RuBERT (rubert-ru-sentiment-rusentiment) for accurate Russian-language sentiment classification.
  • Implements automated topic filters to focus on key Ukraine-related keywords and phrases.
  • Displays aggregated results in an interactive front end, showing polarity trends over time.
  • Supports export of structured JSON data for deeper analysis and cross-referencing with other OSINT sources.

Technical Details

The backend, written in Python (Flask), manages API requests and model inference through Hugging Face Transformers. Data from VKontakte is filtered and processed in real time. The frontend, built in Bootstrap 5 and JavaScript, visualises the sentiment distribution and allows for keyword-based exploration. This modular structure supports expansion to include Telegram or Twitter APIs.

Rationale & Purpose

This project demonstrates how social media monitoring can support open-source intelligence by tracking sentiment shifts in Russian-language platforms. It was designed to examine the tone and framing of news agency narratives about the Ukraine conflict as shared inside Russia. The system illustrates how real-time language data and machine learning can be combined to support early warning and influence analysis.
