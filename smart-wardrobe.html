Smart Wardrobe

Real-World Problem Solving through Computer Vision and Sustainable Design

Year: 2022

Technologies: LAMP Stack (Linux, Apache, MySQL, PHP), C#, AWS Rekognition, Computer Vision

Rationale & Purpose

The Smart Wardrobe project began with an everyday frustration that turned into a technical challenge. Inspired by the visual carousels used on McDonald’s touchscreen kiosks, Alyzande saw the potential for a similar interface to bring order to her “floordrobe” at home. The goal was to create a digital wardrobe that let users see what they owned, plan outfits, and reduce overbuying caused by visual clutter or forgotten garments.

This project represents Alyzande’s pragmatic approach to technology: every feature originated from a tangible personal problem and was developed into a scalable, human-centred solution. The Smart Wardrobe exemplifies how real-world frustration can become a driver for applied innovation, where computer vision and data systems are used not as novelty but as tools to simplify life and promote sustainability.

The system was designed as a full-stack CRUD web application, hosted on a LAMP stack with a MySQL database and a carousel-based front end. It allowed users to catalogue clothing, track usage frequency, and calculate cost per wear. This supported more sustainable fashion decisions and reduced unnecessary purchases. Alyzande then extended the concept with AWS Rekognition, using computer vision to automatically identify and categorise garments from webcam images. This feature allowed the wardrobe to populate itself, bringing AI efficiency into a domestic sustainability tool.

The project also contributed to Alyzande’s MSc research, where Rekognition’s accuracy was compared to emerging academic models in clothing recognition. The results showed that AWS Rekognition performed competitively while being accessible for practical consumer applications.

Key Features

  • Full CRUD web app enabling users to add, edit, and remove wardrobe items.
  • Touchscreen carousel interface for intuitive browsing and outfit planning.
  • Automatic clothing recognition using AWS Rekognition for live garment tagging.
  • Cost-per-wear tracking and sustainability metrics to highlight clothing value and reduce waste.
  • Data-driven design promoting responsible consumer behaviour and minimising fast-fashion impact.

Technical Details

Built using the LAMP stack (Linux, Apache, MySQL, PHP) with a C#-driven front end. Integrated AWS Rekognition for real-time image classification and tagging. The application stored metadata such as brand, colour, and last-worn date to enable analytics and environmental insights. The code structure prioritised scalability, modularity, and the ability to integrate future IoT sensors or wardrobe-mounted tablets.
