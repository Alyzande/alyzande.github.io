Selected Projects

Shadow Fleet Tracker — Maritime Intelligence Platform (2025)

Developed a real-time maritime intelligence system that detects and visualises suspicious vessel activity using global AIS (Automatic Identification System) data. The platform combines a Python geospatial backend (Flask, GeoPandas, Shapely) with a Leaflet.js dashboard to monitor vessels operating under “flags of convenience” and those loitering near undersea infrastructure.

Key features: Real-time AIS ingestion, anomaly detection, geofencing alerts, live vessel tracking, modular architecture for satellite and sanctions data integration.
