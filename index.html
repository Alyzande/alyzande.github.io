Shadow Fleet Tracker — Maritime Intelligence Platform (2025) (In Progress)

A real-time maritime intelligence platform that detects and visualises suspicious vessel activity using global AIS (Automatic Identification System) data. Combines a Python geospatial backend (Flask, GeoPandas, Shapely) with a Leaflet.js dashboard to monitor vessels near critical undersea infrastructure.

Key features: Real-time AIS ingestion and anomaly detection, geofencing alerts, interactive dashboard, modular integration with satellite and sanctions data.

Technologies: Python, Flask/FastAPI, WebSockets, GeoPandas, Leaflet.js, JavaScript, AISStream.io, TeleGeography GeoJSON.
