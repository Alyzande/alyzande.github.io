Technical Details

The backend is written in Python (Flask), with a RESTful API that connects to the RuBERT sentiment model hosted on Hugging Face for text inference. The frontend uses Bootstrap 5 and JavaScript for an interactive experience, with colour-coded visual feedback and live news integration via NewsAPI. Results are returned as structured JSON with polarity and confidence scores.

Rationale & Purpose

This project was designed to demonstrate applied NLP for OSINT and sentiment analysis in foreign-language contexts. It supports understanding of Russian-language media tone, public sentiment, and disinformation narratives ‚Äî areas of increasing importance for language-enabled intelligence analysis and data-driven policy insight.
