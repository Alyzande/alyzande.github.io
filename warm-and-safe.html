Warm & Safe

Mapping Community Warm Spaces Across the UK with Accessible Web Design

Year: 2022–2023

Technologies: HTML, CSS, JavaScript, React, Node.js, Express, MongoDB (Azure), Leaflet.js, OpenStreetMap API, JWT Authentication

Overview

Warm & Safe is a humanitarian web application built to help people locate nearby “warm banks”, public spaces offering warmth, safety, and social connection during the UK energy crisis. The platform provided an interactive national map of over 3,500 verified community locations, including libraries, churches, and sports centres. Each entry displayed real-time accessibility details such as heating, seating, toilets, children’s play areas, and free Wi-Fi.

Key Features

  • Interactive map built with Leaflet.js and OpenStreetMap API, showing warm spaces across the UK.
  • Dynamic markers indicated open or closed status and facilities on hover or tap.
  • Data stored in a MongoDB database hosted on Azure, managed through a secure Node.js / Express backend.
  • Authenticated community and council users could add, edit, or remove listings through a React front end with JWT-based access control.
  • Accessible design with clear colour contrast, ARIA labels, and screen-reader-friendly structure.

Technical Details

Warm & Safe was built using a full MERN stack architecture with secure CRUD functionality. Approved users such as council or faith-group representatives could add, update, or remove listings directly through the web interface when opening hours or facilities changed. The system used JSON Web Tokens (JWT) for authentication, a Node.js/Express backend, and a MongoDB database hosted on Azure. Map data was rendered dynamically using Leaflet.js and the OpenStreetMap API, showing real-time location details with mouseover pop-ups for accessibility. Accessibility features included ARIA roles, keyboard navigation, and clear colour contrast to ensure screen-reader compatibility.

Rationale & Purpose

Warm & Safe was inspired by widespread reports of people unable to afford heating during the UK’s energy crisis. The idea was to create a national network of free public warm spaces, similar to food banks, to provide dignity and safety for vulnerable residents. Alyzande contributed as full-stack developer and data architect, responsible for database structure, API integration, and interface accessibility. Working in an agile team, she collaborated via Slack and Discord with front-end developer Bradley Ashton, ensuring inclusive and ethical design. The project avoided commercial listings, ensuring every space was genuinely open to all. By winter 2022, the site was featured on BBC News Online and BBC Local Radio, helping thousands of people find warm, safe community spaces across the UK.
